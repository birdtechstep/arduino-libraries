This is a library for the Ayarafun ST7789H2 display products

This library works with the TFT LCD 1.22 Inchs. [TC122B-8171A-N] (SPI)
1.22"inch round lcd display 240x204 dots
  ----> http://www.ayarafun.com
 
Check out the links above for our tutorials and wiring diagrams.
These displays use SPI to communicate, 6 or 7 pins are required
to interface (RST is optional).

Ayarafun invests time and resources providing this open source code,
please support Ayarafun and open-source hardware by purchasing
products from Ayarafun!

To download. click the DOWNLOADS button in the top right corner, rename the uncompressed folder Ayarafun_ST7789H2. Check that the Ayarafun_ST7789H2 folder contains Ayarafun_ST7789H2.cpp and Ayarafun_ST7789H2.

Place the Ayarafun_ST7789H2 library folder your arduinosketchfolder/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.
